Detailed Description

Widget for slider input for textual display and direct entry of the value.

Inherits a large part of its functionality from the CForge::InputTextWidget.

◆ getFloatValue()

float CForge::InputSliderWidget_Text::getFloatValue ( )

Returns the entered text as float value.

As this sub-widget inherits from CForge::InputTextWidget, the same rules for updating the value as mentioned in CForge::InputTextWidget::onKeyPress apply. Namely, this function will only return the last valid input, which may not be what is currently entered.

◆ setFloatValue()

void CForge::InputSliderWidget_Text::setFloatValue ( float  value)

Sets the value and displayed text to the one passed in.

Internally converts the value to a rounded representation with the amount of decimals specified in CForge::InputSliderStyle::RoundLabelNumber. Note that only the visuals are rounded, the float value is set as is.

◆ validateInput()

bool CForge::InputSliderWidget_Text::validateInput ( )
overrideprivatevirtual

Checks if input can be read as float and sets value if it does.

This validation function tries to parse the given input as float value using std::stof. If it succeeds, the value is then submitted to InputSliderWidget::setValue to be used as new value of the widget.

One thing to note might be that this function here doesn't actually set the value of this here text widget. That is instead left to InputSliderWidget::setValue as only that function performs the limit and step size checks. It will then call setFloatValue.
